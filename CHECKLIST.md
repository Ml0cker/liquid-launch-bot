# Чек-лист развертывания Liquid Protocol Launch Bot

## Предварительная подготовка

- [ ] Установлен Docker Desktop / Docker Engine
- [ ] Установлен Docker Compose
- [ ] Есть Telegram аккаунт
- [ ] Есть доступ в интернет

## Получение учетных данных

- [ ] Создан Telegram Bot через @BotFather
- [ ] Скопирован TELEGRAM_BOT_TOKEN
- [ ] Создан публичный канал или чат
- [ ] Получен TELEGRAM_CHAT_ID
- [ ] Бот добавлен в канал с правами администратора

## Клонирование и конфигурация

- [ ] Клонирован репозиторий
- [ ] Создан файл .env из .env.example
- [ ] Заполнены все обязательные переменные:
  - [ ] TELEGRAM_BOT_TOKEN
  - [ ] TELEGRAM_CHAT_ID
  - [ ] BASE_RPC_URL (опционально, есть значение по умолчанию)
  - [ ] START_BLOCK (опционально, есть значение по умолчанию)
- [ ] Файл .env добавлен в .gitignore
- [ ] Файл .env НЕ закоммичен в репозиторий

## Решение проблем

### Если бот не отправляет сообщения

- [ ] Проверен TELEGRAM_BOT_TOKEN в .env
- [ ] Проверен TELEGRAM_CHAT_ID в .env
- [ ] Проверено, что бот добавлен в канал
- [ ] Проверено, что у бота есть права администратора
- [ ] Проверены логи на ошибки Telegram API
- [ ] Попробовано отправить тестовое сообщение вручную

### Если нет подключения к RPC

- [ ] Проверено интернет соединение
- [ ] Проверен BASE_RPC_URL в .env
- [ ] Попробован альтернативный RPC
- [ ] Проверены логи на ошибки подключения

### Если изображения не загружаются

- [ ] Проверен IPFS_GATEWAY в .env
- [ ] Попробован альтернативный IPFS gateway
- [ ] Проверены логи на ошибки загрузки

### Если БД растет слишком быстро

- [ ] Проверено, что дубликаты не обрабатываются
- [ ] Проверено, что START_BLOCK установлен правильно
- [ ] Рассмотрена очистка старых записей
